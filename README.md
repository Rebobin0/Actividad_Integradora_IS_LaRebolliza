## Objetivo de la actividad

Implementar una aplicación tipo FULLSTACK. El propósito puede ser la gestión de productos y verificar la disponibilidad o existencia.

- El frontend con React y react router DOM, utilizando Node.js
- El Backend mediante el servidor Express, usando el PERN Stack
- Separar el Cliente del Server
- Hacer uso de los CORS
- Adicionalmente agregar Testing con Jest
- Usar Morgan para el Login
- Documentar el API con Swagger 
- Usar supertest para escribir el testing del código 
- Usar Sequelize para las consultas hacia la Rest API 
- Express validator para validar las entradas de datos
